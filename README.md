# 🌌 Overview

**AI Hand-Tracked Particle Universe** is an interactive browser experience that combines **Artificial Intelligence**, **Computer Vision**, **Three.js**, and **Particle Physics**.

Instead of using a mouse or keyboard, users interact naturally with thousands of glowing particles using only hand gestures captured by their webcam.

Powered by **MediaPipe Hands**, the application recognizes gestures in real time and transforms them into immersive 3D interactions.

## ✨ Interactive Gestures

### 👋 Hand Movement

Move your hand to repel nearby particles.

Particles react naturally with realistic force.

---

### 🤏 Pinch Gesture

Bring your thumb and index finger together.

This controls camera zoom.

- Zoom In
- Zoom Out
- Smooth Animation

---

### ✊ Fist Gesture

Close your hand into a fist.

Instantly transforms the particle system into another shape.
